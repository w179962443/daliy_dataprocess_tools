## 🚀 使用流程

### 安装 (5 分钟)

```bash
cd d:\daliy_dataprocess_tools\realtime_transcriber
pip install -r requirements.txt
```

### 启动 (选一种)

```bash
# Web版本
python app.py
# 访问 http://localhost:5000

# 桌面版本
python app_pyqt.py

# Windows启动
run.bat
```

### 使用

```
1. 选择音源和语言
2. 点击"开始转录"
3. 说话或播放音频
4. 实时查看转录结果
5. 点击"停止转录"
6. 文件自动保存
```
