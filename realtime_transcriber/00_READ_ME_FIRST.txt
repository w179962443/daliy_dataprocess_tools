## 🔧 主要配置

编辑 `config.py` 修改:

```python
# 模型选择
'model_name': 'base'       # tiny/base/small/medium/large

# 语言设置
'language': 'auto'         # auto/zh/en/ja/ko/fr/de等

# 转录间隔
'transcribe_interval': 2   # 秒数

# 文件保存位置
'output_dir': 'recordings' # 目录路径
```
